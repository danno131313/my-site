About me

Hi, my name is Daniel Kavanagh and I'm a software engineer specializing in full stack web development in Bellingham, WA.

Programming is one of my main passions, right alongside music. I've always loved building things, from houses made of Lego when I was little, to the multi-part webapps incorperating different frameworks and libraries/tools I build now. The satisfaction from building a functioning app with clean, concise code is the entire reason I program; it doesn't matter what I'm working on, I'll have fun working on it regardless!
