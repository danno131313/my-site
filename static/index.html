About me

Hi, my name is Daniel Kavanagh and I'm a software engineer from Bellingham, WA.

Ever since I was little, I've always loved building things and solving problems, from houses made of Lego when I was 5, to the complex web apps I build today. The satisfaction from creating clean and concise code is the entire reason I program; it doesn't matter what I'm working on, I'll always have fun!

The first time I was introduced to coding was when I was 13. I wanted to make a video game character stronger, so I poked around the files and started changing numbers. The code was so fascinating to me, and it looked incredibly complicated to write, but my tinkering worked and I achieved my desired results! From that day forward, tinkering with software was always one of my passions. I quickly installed Linux on my computer, rooted my phone, and have been enjoying the "break it, fix it" cycle ever since.
